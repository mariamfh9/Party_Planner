
# Party Planner

### Overview

An App that keeps track of anything &amp; everything party! Never forget anything on your todo list!


### Installation Instructions

Run bundle install like so: 

```
$ bundle install 

```

### Usage

Type the following to startup the rails server: 

```
$ rails s
```
